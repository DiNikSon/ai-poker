          ## Быстрый старт

          1. **Создайте файл бота** `bot.js`
          2. 2. **Экспортируйте конфигурацию и функцию turn**
             3. 3. **Загрузите на платформу** и присоединяйтесь к турниру
               
                4. ```javascript
                   module.exports = {
                     config: { /* ... */ },
                     turn: ({ game, history, memory }) => { /* ... */ }
                   }
                   ```

                   ## Правила Texas Hold'em

                   ### Цель игры
                   Выиграть фишки соперников, собрав лучшую комбинацию из пяти карт.

                   ### Раунды торговли

                   1. **Pre-flop** - После раздачи двух закрытых карт каждому игроку
                   2. 2. **Flop** - После открытия первых трёх общих карт
                      3. 3. **Turn** - После открытия четвёртой общей карты
                         4. 4. **River** - После открытия пятой общей карты
                            5. 5. **Showdown** - Вскрытие карт (если игроки дошли сюда)
                              
                               6. ### Возможные действия в ваш ход
                              
                               7. - **Check** (pасход) - Остаться в игре без ставки (доступно только если никто не ставил)
                                  - - **Bet** (ставка) - Сделать первую ставку
                                    - - **Call** (уравнять) - Уравнять текущую ставку
                                      - - **Raise** (рейз) - Увеличить ставку
                                        - - **Fold** (сброс) - Выйти из раунда (потеряете вложенные фишки)
                                          - - **All-in** (олл-ин) - Поставить все оставшиеся фишки
                                           
                                            - ### Комбинации (старшинство от высокого к низкому)
                                           
                                            - | Комбинация | Описание | Пример |
                                            - |-----------|---------|--------|
                                            - | Royal Flush | A-K-Q-J-10 одной масти | ♠A ♠K ♠Q ♠J ♠10 |
                                            - | Straight Flush | Пять карт подряд одной масти | ♥9 ♥8 ♥7 ♥6 ♥5 |
                                            - | Four of a Kind | Четыре карты одного ранга | ♠K ♥K ♣K ♦K |
                                            - | Full House | Тройка + пара | ♠7 ♥7 ♣7 ♠2 ♥2 |
                                            - | Flush | Пять карт одной масти | ♣2 ♣4 ♣6 ♣8 ♣T |
                                            - | Straight | Пять карт подряд | ♠5 ♥6 ♣7 ♦8 ♠9 |
                                            - | Three of a Kind | Тройка карт одного ранга | ♠J ♥J ♣J ♦2 ♣3 |
                                            - | Two Pair | Две пары | ♠4 ♥4 ♣8 ♦8 ♠K |
                                            - | One Pair | Одна пара | ♠9 ♥9 ♣2 ♦5 ♣7 |
                                            - | High Card | Старшая карта | ♠A ♥K ♣Q ♦J ♣9 |
                                           
                                            - ### Структура игры
                                           
                                            - - **Small blind** - обязательная малая ставка (игрок слева от дилера)
                                              - - **Big blind** - обязательная большая ставка (игрок слева от small blind)
                                                - - **Минимум 2 игрока**, максимум 8 игроков за столом
                                                  - - **Раунд завершается**, когда все игроки либо выбыли, либо уравняли ставку

                                                    ## Структура бота

                                                    Ваш файл `bot.js` должен экспортировать объект с двумя частями:

                                                    ### 1. Конфигурация (config)

                                                    ```javascript
                                                    config: {
                                                      name: "Имя вашего бота",                                    // Строка (обязательно)
                                                      image_url: "https://example.com/avatar.jpg",               // URL аватара (обязательно)
                                                      author_name: "Иван Иванов",                                // Имя автора (обязательно)
                                                      description: "Краткое описание стратегии бота...",         // Строка (обязательно)
                                                      win_say: [
                                                        "Я - король покера! 👑",                                 // Фраза за 1 место
                                                        "Неплохо для второго места",                             // Фраза за 2 место
                                                        "Третье место - не худший результат"                     // Фраза за 3 место
                                                      ],                                                          // Массив (обязательно)
                                                      lose_say: "В следующий раз повезёт 🍀"                    // Фраза при выбытии (обязательно)
                                                    }
                                                    ```

                                                    **Параметры:**

                                                    | Параметр | Тип | Описание | Лимит |
                                                    |----------|-----|---------|-------|
                                                    | `name` | String | Отображаемое имя бота | max 50 символов |
                                                    | `image_url` | String | URL изображения аватара | Должен быть доступен по HTTPS |
                                                    | `author_name` | String | Ваше имя или никнейм | max 100 символов |
                                                    | `description` | String | Описание стратегии бота | max 500 символов |
                                                    | `win_say[0]` | String | Победная фраза за 1 место | max 100 символов |
                                                    | `win_say[1]` | String | Фраза за 2 место | max 100 символов |
                                                    | `win_say[2]` | String | Фраза за 3 место | max 100 символов |
                                                    | `lose_say` | String | Фраза при выбытии | max 100 символов |

                                                    ### 2. Функция turn

                                                    ```javascript
                                                    turn: ({ game, history, memory }) => {
                                                      // Ваша логика принятия решения
                                                      game.fold()
                                                      return memory
                                                    }
                                                    ```

                                                    **Параметры функции:**

                                                    - **game** - Объект с методами для совершения действий (см. [API методы](#api-методы))
                                                    - - **history** - История хода игры (карты, ставки, действия)
                                                      - - **memory** - Объект для сохранения состояния между ходами (персистентная память)
                                                       
                                                        - **Возвращаемое значение:** Обновленный объект `memory`

                                                        ## API методы

                                                        ### Методы действия (game)

                                                        #### game.fold()
                                                        Выйти из раунда, потеряв все вложенные фишки в этот раунд.

                                                        ```javascript
                                                        game.fold()
                                                        ```

                                                        #### game.check()
                                                        Остаться в игре без ставки (доступно только если никто не повышал ставку).

                                                        ```javascript
                                                        game.check()
                                                        ```

                                                        #### game.call()
                                                        Уравнять текущую ставку соперника.

                                                        ```javascript
                                                        game.call()
                                                        ```

                                                        #### game.bet(amount)
                                                        Сделать ставку указанного размера (первая ставка в раунде).

                                                        ```javascript
                                                        game.bet(100)  // Ставка 100 фишек
                                                        ```

                                                        #### game.raise(amount)
                                                        Увеличить ставку на указанную сумму относительно текущей ставки.

                                                        ```javascript
                                                        game.raise(200)  // Повысить на 200 фишек
                                                        ```

                                                        #### game.all_in()
                                                        Поставить все оставшиеся фишки.

                                                        ```javascript
                                                        game.all_in()
                                                        ```

                                                        #### game.say(message)
                                                        Вывести сообщение в чат игры (видно всем).

                                                        ```javascript
                                                        game.say("Я вижу вашу ставку!")
                                                        ```

                                                        ### Объект History

                                                        Содержит информацию об истории игры:

                                                        ```javascript
                                                        history = {
                                                          round: 1,                              // Номер раунда (1, 2, 3...)
                                                          stage: "flop",                         // Этап: "pre-flop", "flop", "turn", "river"
                                                          my_cards: ["AS", "KH"],                // Ваши две карты
                                                          community_cards: ["2C", "3D", "4H"],   // Общие карты на столе
                                                          players: [
                                                            {
                                                              id: "player1",
                                                              stack: 1500,                       // Количество фишек
                                                              is_me: false,
                                                              is_active: true,                   // Остаётся ли в игре
                                                              bet_amount: 100,                   // Его текущая ставка
                                                              position: "button",                // "button", "small_blind", "big_blind", "other"
                                                              cards_visible: false
                                                            },
                                                            // ... остальные игроки
                                                          ],
                                                          current_bet: 200,                      // Текущая ставка для уравнивания
                                                          pot: 500,                              // Общий банк
                                                          my_bet: 50,                            // Ваша ставка в текущем раунде
                                                          available_actions: ["call", "raise", "fold"]  // Доступные действия
                                                        }
                                                        ```

                                                        **Значения карт:**

                                                        Карты кодируются строками из двух символов:
                                                        - **Ранг**: `A`, `K`, `Q`, `J`, `T`, `9`, `8`, `7`, `6`, `5`, `4`, `3`, `2`
                                                        - - **Масть**: `S` (♠ пики), `H` (♥ черви), `D` (♦ бубны), `C` (♣ трефы)
                                                         
                                                          - Примеры: `AS` (туз пик), `KH` (король черв), `2D` (двойка бубнов)
                                                         
                                                          - ---

                                                          ### Объект Memory

                                                          Объект для сохранения информации между ходами. Используйте его для отслеживания стратегии:

                                                          ```javascript
                                                          turn: ({ game, history, memory }) => {
                                                            // Инициализируем память при первом ходе
                                                            if (!memory.stats) {
                                                              memory.stats = {
                                                                wins: 0,
                                                                losses: 0,
                                                                big_wins: []
                                                              }
                                                            }

                                                            // Используем сохраненные данные
                                                            console.log(`Побед: ${memory.stats.wins}`)

                                                            // Обновляем память
                                                            memory.stats.wins++

                                                            return memory  // Важно! Всегда возвращайте обновленный memory
                                                          }
                                                          ```

                                                          **Ограничения:**
                                                          - Максимум 10 KB данных per память
                                                          - - Сохраняется между раундами одного турнира
                                                            - - Сбрасывается при начале нового турнира

                                                              ## Примеры

                                                              ### Пример 1: Простой агрессивный бот

                                                              ```javascript
                                                              module.exports = {
                                                                config: {
                                                                  name: "Агрессор",
                                                                  image_url: "https://fastly.picsum.photos/id/237/200/200.jpg",
                                                                  author_name: "Иван Иванов",
                                                                  description: "Агрессивная стратегия: часто повышает ставки",
                                                                  win_say: [
                                                                    "Агрессия победила! 💪",
                                                                    "На следующий раз буду ещё агрессивнее",
                                                                    "И третьего раза не будет"
                                                                  ],
                                                                  lose_say: "Иногда агрессия не помогает"
                                                                },

                                                                turn: ({ game, history, memory }) => {
                                                                  const available = history.available_actions

                                                                  // Если наша карта сильная, повышаем
                                                                  if (history.my_cards.some(card => card[0] === 'A')) {
                                                                    if (available.includes('raise')) {
                                                                      game.raise(100)
                                                                    } else if (available.includes('bet')) {
                                                                      game.bet(100)
                                                                    }
                                                                  }
                                                                  // Иначе просто уравниваем или выходим
                                                                  else if (available.includes('call')) {
                                                                    game.call()
                                                                  } else {
                                                                    game.fold()
                                                                  }

                                                                  return memory
                                                                }
                                                              }
                                                              ```

                                                              ### Пример 2: Консервативный бот с памятью

                                                              ```javascript
                                                              module.exports = {
                                                                config: {
                                                                  name: "Консервативчик",
                                                                  image_url: "https://fastly.picsum.photos/id/238/200/200.jpg",
                                                                  author_name: "Пётр Петров",
                                                                  description: "Консервативная тактика с анализом оппонентов",
                                                                  win_say: [
                                                                    "Терпение - путь к победе! 🎯",
                                                                    "Следующий раз буду ещё терпеливее",
                                                                    "Опыт - лучший учитель"
                                                                  ],
                                                                  lose_say: "Даже консерватор может проиграть"
                                                                },

                                                                turn: ({ game, history, memory }) => {
                                                                  // Инициализируем статистику
                                                                  if (!memory.opponent_bets) {
                                                                    memory.opponent_bets = {}
                                                                  }

                                                                  const my_cards = history.my_cards
                                                                  const is_strong_hand = (
                                                                    my_cards.some(card => card[0] === 'A') ||
                                                                    my_cards.some(card => card[0] === 'K')
                                                                  )

                                                                  // Сильная рука в начале раунда
                                                                  if (history.stage === 'pre-flop' && is_strong_hand) {
                                                                    if (history.available_actions.includes('raise')) {
                                                                      game.raise(50)
                                                                    } else if (history.available_actions.includes('call')) {
                                                                      game.call()
                                                                    }
                                                                  }
                                                                  // Слабая рука
                                                                  else if (history.current_bet > history.my_bet) {
                                                                    game.fold()
                                                                  }
                                                                  // Нейтральная ситуация
                                                                  else {
                                                                    if (history.available_actions.includes('check')) {
                                                                      game.check()
                                                                    } else if (history.available_actions.includes('call')) {
                                                                      game.call()
                                                                    } else {
                                                                      game.fold()
                                                                    }
                                                                  }

                                                                  return memory
                                                                }
                                                              }
                                                              ```

                                                              ### Пример 3: Адаптивный бот

                                                              ```javascript
                                                              module.exports = {
                                                                config: {
                                                                  name: "Адаптер",
                                                                  image_url: "https://fastly.picsum.photos/id/239/200/200.jpg",
                                                                  author_name: "Александра Смирнова",
                                                                  description: "Анализирует оппонентов и адаптирует стратегию",
                                                                  win_say: [
                                                                    "Адаптация работает! 🧬",
                                                                    "Хороший результат, но есть куда расти",
                                                                    "Медаль, но не золотая"
                                                                  ],
                                                                  lose_say: "Даже адаптация имеет пределы"
                                                                },

                                                                turn: ({ game, history, memory }) => {
                                                                  // Отслеживаем агрессивность оппонентов
                                                                  if (!memory.game_data) {
                                                                    memory.game_data = {
                                                                      round: 0,
                                                                      aggressive_opponents: [],
                                                                      tight_opponents: []
                                                                    }
                                                                  }

                                                                  memory.game_data.round++

                                                                  // Оценка силы руки (упрощенная)
                                                                  const card_values = { 'A': 14, 'K': 13, 'Q': 12, 'J': 11, 'T': 10 }
                                                                  const hand_strength = Math.max(
                                                                    card_values[history.my_cards[0][0]] || parseInt(history.my_cards[0][0]),
                                                                    card_values[history.my_cards[1][0]] || parseInt(history.my_cards[1][0])
                                                                  )

                                                                  // Адаптивная стратегия
                                                                  if (hand_strength >= 12) {
                                                                    // Сильная рука - активно играем
                                                                    if (history.available_actions.includes('raise')) {
                                                                      game.raise(80)
                                                                    } else {
                                                                      game.call()
                                                                    }
                                                                  } else if (hand_strength >= 8) {
                                                                    // Средняя рука - осторожно
                                                                    if (history.available_actions.includes('check')) {
                                                                      game.check()
                                                                    } else {
                                                                      game.call()
                                                                    }
                                                                  } else {
                                                                    // Слабая рука - выходим
                                                                    game.fold()
                                                                  }

                                                                  return memory
                                                                }
                                                              }
                                                              ```

                                                              ## Советы по стратегии

                                                              ### 1. Позиция важна
                                                              - **Button и близко к Button** - можно играть шире рук
                                                              - - **Ранняя позиция (Big Blind)** - играйте только сильные руки
                                                               
                                                                - ### 2. Стартовые руки (Pre-flop)
                                                                - Играйте только эти руки в ранней позиции:
                                                                - - Любые карты Ace (A-x)
                                                                  - - Пары: JJ+
                                                                    - - Сильные карты: KQ, AK
                                                                     
                                                                      - ### 3. Размер ставок
                                                                      - - **Малые ставки** - привлекают больше игроков (для слабых рук нежелательно)
                                                                        - - **Большие ставки** - выселяют слабых игроков (хорошо для сильных рук)
                                                                         
                                                                          - ### 4. Фазы игры
                                                                          - - **Pre-flop** - самая важная фаза, здесь определяется сильная рука
                                                                            - - **Flop** - у вас есть 5 карт, проще оценить потенциал
                                                                              - - **Turn & River** - последний момент для решений
                                                                               
                                                                                - ### 5. Банк-роллинг
                                                                                - - Не рискуйте всеми фишками на спорных решениях
                                                                                  - - Сохраняйте стек для получше возможностей
                                                                                   
                                                                                    - ### 6. Чтение оппонентов (в памяти)
                                                                                    - Отслеживайте в `memory`:
                                                                                    - - Кто часто поднимает ставки?
                                                                                      - - Кто часто фолдит?
                                                                                        - - На какой позиции они играют?
                                                                                         
                                                                                          - ### 7. Математика банка
                                                                                          - - **Pot odds**: Если для call нужно 50 фишек, а в банке 200, odds = 4:1
                                                                                            - - Если вероятность выиграть > 20% (1 из 5), то call выгоден

                                                                                              ## Часто задаваемые вопросы

                                                                                              ### Можно ли использовать случайные числа?
                                                                                              Да! `Math.random()` доступен:
                                                                                              ```javascript
                                                                                              if (Math.random() > 0.5) {
                                                                                                game.bet(50)
                                                                                              }
                                                                                              ```

                                                                                              ### Какой максимальный размер памяти?
                                                                                              10 KB - не забывайте очищать старые данные

                                                                                              ### Как долго может работать функция turn?
                                                                                              Максимум 1 секунда на ход. После истечения времени будет выполнено действие по умолчанию (fold)

                                                                                              ### Что если мои фишки закончились?
                                                                                              Вы автоматически выбываете из турнира. Сработает `lose_say`

                                                                                              ### Можно ли проверить карты соперников?
                                                                                              Нет - это скрытая информация. В `history.community_cards` видны только общие карты

                                                                                              | Параметр | Лимит | Примечание |
                                                                                              |----------|-------|-----------|
                                                                                              | Размер конфига | 1 KB | Проверяется при загрузке |
                                                                                              | Размер памяти | 10 KB | Per bot, per tournament |
                                                                                              | Время на ход | 1000 ms | Превышение = fold |
                                                                                              | Длина сообщения в say() | 200 символов | Сокращается при превышении |
                                                                                              | Количество игроков | 2-8 | На одном столе |
                                                                                              | Минимальная ставка | 1 фишка | Или Big Blind |
                                                                                              | Максимальная ставка | Весь стек | All-in |
